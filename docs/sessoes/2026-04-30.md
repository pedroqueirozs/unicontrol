# Sessão 2026-04-30

`feat/cadastros`

## O que foi feito

### Dashboard — correções
- Atalhos atualizados: dois cards de pendências (`/suppliers-pending` + `/customers-pending`) substituídos por um único card "Pendências" → `/pendencias`
- Cálculo de mercadorias atrasadas corrigido: agora compara com `startOf("day")` em vez de `new Date()` (impedia que envios com previsão para hoje fossem marcados como atrasados)

### Cadastros — melhorias
- **Código auto-gerado**: removido campo manual de código; sistema gera automaticamente `C-001`, `C-002`... para clientes e `F-001`, `F-002`... para fornecedores com base no maior número existente em memória
- **CNPJ/CPF obrigatório**: campo passou de opcional para obrigatório
- **Máscara automática CNPJ/CPF**: detecta tipo pelo número de dígitos (≤11 = CPF, >11 = CNPJ) e aplica formatação enquanto digita
- **Máscara automática CEP**: formata como `00000-000`
- **Busca por CNPJ**: DataGrid e todos os dropdowns de busca nos módulos passaram a filtrar por nome ou CNPJ/CPF (antes era por código)
- Mesmo CNPJ pode ter múltiplos cadastros (endereços diferentes)

### Mercadorias Enviadas — novas funcionalidades
- **Cliente obrigatório (RN-19)**: não é mais possível salvar sem vincular um cliente do cadastro
- Nome, cidade e UF são preenchidos automaticamente e bloqueados para edição manual
- **Fix de edição**: ao clicar em editar, o cliente é restaurado a partir do `clientId` salvo
- **Flag de atenção**: ícone de bandeira em cada linha; clique ativa/desativa; linha ganha fundo vermelho claro; persiste no Firestore
- **Códigos de rastreio**: campo dinâmico (campo + botão Adicionar); múltiplos códigos por envio; salvo como `trackingCodes: string[]` no Firestore; campo pendente no input é auto-incluído no save
- **Copiar código**: botão de cópia em cada código de rastreio no modal de detalhe; feedback visual de ✓ por 2 segundos

### Pendências — correção de nomenclatura
- Campos renomeados de `contactId`/`contactCode` (genérico) para:
  - `customers_pending`: `clientId` + `clientCode`
  - `suppliers_pending`: `supplierId` + `supplierCode`
